# Governance

## Project Ownership

**Primary Owner**: Joshua Jewell ([@JoshuaJewell](https://github.com/JoshuaJewell))

## Decision Making

This is a family co-developed project. Decisions are made collaboratively between the primary owner and admin.

- **Gameplay design**: Led by the primary owner
- **Technical architecture**: Led by the admin
- **Release decisions**: Joint agreement required

## Contributions

External contributions are welcome via pull requests. See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

All contributions must:
1. Be compatible with AGPL-3.0-or-later
2. Follow the project's coding standards (see `.editorconfig`)
3. Pass CI checks before merge
4. Be reviewed by at least one maintainer
