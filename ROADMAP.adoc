= Airborne Submarine Squadron Roadmap

== Current Stage

The Gossamer edition now boots a loader, shows retro battle art, draws a live solar-map HUD, and connects the digital instruments with the orbital menu. The system is deterministic, network-ready, and already records leaderboard runs plus damage diagrams in real time.

== Milestones

=== v2.0 — Orbital Gossamer (released)
* [x] Gossamer renderer with KITT-style HUD and digital speedometer/accelerometer.
* [x] Depth charges, torpedoes, missiles, and afterburner toggles.
* [x] Orbit menu with warp hotkeys (F + Enter) and direct 1‑5 destinations, sun hazard burnout, and cave messages.
* [x] Leaderboard persistence, compact legend, and pause settings with sub-skin tuning.

=== v2.1 — Customization Layer (shipped 2026-04-16)
* [x] Colour swatch preview + R/B/P skin-preset hotkeys in the pause menu.
      Spectrum hue still tunable via `,` / `.`, and a live 5-part swatch
      (hull/wings/tower/nose/porthole) renders in the settings panel.
* [x] Damage map refinement: depleted parts flicker/blur in the diagram,
      unused penalty functions (back damage, front control, hull buoyancy)
      are wired into movement, and hull damage bleeds buoyancy underwater.
* [x] Warp preview: solar minimap now shows destination name, hazard
      warnings, live range, and the orbit menu lists all warp hotkeys
      with per-planet accent/sky/enemy colour swatches.
* [x] Pride/Rainbow/Spectrum skins documented in README controls section
      with persistence details and keybind cheatsheet.
* [x] Controls/keybinds extracted into `gossamer/controls.js` — single
      source of truth for key state, DEFAULT_KEYBINDS, rebind actions,
      and the control scheme legend.

=== v3.0 — Multi-platform Deployment
* [ ] Cross-platform packaging: Linux (deb/rpm), Kinoite/Silverblue, Windows, macOS, Android/iOS webview, Minix, RISC-V, Lua/Node wrappers.
* [ ] Launcher/issuer automation (`launcher.sh`, `integ.sh`, `doinstall`), CICD health, and desktop/start menu integration.
* [ ] AffineScript build chain checks with prebuilt WASM fallback messaging.
