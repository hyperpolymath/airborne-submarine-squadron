= Airborne Submarine Squadron (ASS)

image:https://img.shields.io/badge/License-Palimpsest--MPL%201.0-blue.svg[License: Palimpsest-MPL 1.0,link=https://github.com/hyperpolymath/palimpsest-license]

Airborne Submarine Squadron is a browser-forward take on the classic Sopwith-style arcade, rebuilt on the Gossamer launcher and wired to a Palimpsest-MPL licensed stack. The current build layers deterministic simulation, cinematic HUD telemetry, orbital warp transitions, depth-charge weapons, and leaderboard persistence inside a retro-inspired visual shell.

== Status

This repo now hosts a playable Gossamer frontend that wraps the AffineScript/WASM core and the launcher/installer ecosystem. The current goal is finishing the orbit/depth-charge gameplay loop and stabilizing builds, installs, and multi-platform deploys.

== Enhanced Integration System (NEW)

The game now features an advanced, cross-platform integration system:

* **Reflective Architecture**: Self-aware installation with platform detection
* **Homoiconic Design**: Self-describing configuration and metadata
* **Self-Healing**: Automatic file verification and backup restoration
* **Fault Tolerance**: Graceful degradation on unsupported platforms
* **Cross-Platform**: Linux, Windows (partial), RISC-V, Minix support
