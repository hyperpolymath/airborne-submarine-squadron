# Airborne Submarine Squadron

[![License: AGPL-3.0-or-later](https://img.shields.io/badge/License-AGPL--3.0--or--later-blue.svg)](https://www.gnu.org/licenses/agpl-3.0.html)

A browser-forward take on the classic Sopwith-style arcade game built with AffineScript/WASM and the Gossamer launcher.

## Installation

```bash
npm install airborne-submarine-squadron
```

## Quick Start

```bash
# Build the WASM
npm run build

# Run the game
npm start

# Or use the launcher
npm run launch
```

## Features

- Free-flight submarine with accelerometer/afterburner
- Weapon triad: torpedoes, missiles, and depth charges
- Digital Knight Rider-style HUD
- Solar map with warp hotkeys
- Leaderboard tracking
- Cross-platform support (Linux, Windows, RISC-V, Minix)
